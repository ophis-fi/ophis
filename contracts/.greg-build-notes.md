# contracts/ build notes

Upstream: cowprotocol/contracts (see `.greg-upstream` for pinned SHA — currently c94c595).

## Toolchain

- **Package manager:** pnpm (NOT yarn). The contracts repo declares no `packageManager` field and ships `yarn.lock`, but the parent monorepo (`/Users/scep/greg/package.json`) declares `pnpm@9.12.0`. Yarn 1.22 in this directory misreads the parent's declaration as "yarn@pnpm@9.12.0" and aborts. pnpm respects the existing `yarn.lock` content via its own resolver.
- **Install command:** `pnpm install --ignore-workspace` (the `--ignore-workspace` flag is required so pnpm does not treat `contracts/` as part of the parent workspace; without it pnpm installs to `/Users/scep/greg/node_modules` instead of `contracts/node_modules`).
- **Build command:** `pnpm exec hardhat compile` (the upstream `yarn build` script also runs `tsc`, but for deploy purposes only the Solidity build is needed).
- **Deploy command:** `pnpm exec hardhat deploy --network <network>` (dispatches to `src/deploy/00*_*.ts` via `hardhat-deploy`).

## Artifacts

- Solidity output: `contracts/build/artifacts/src/contracts/**`
- Cache: `contracts/build/cache/`

## Node version

Tested with Node 25.9.0. Hardhat 2.28.6 prints a "not supported" warning for Node >22 but compile + deploy work correctly. If issues arise, downgrade to Node 22 LTS.

## Deploy script behaviour (verified on hardhat network)

`pnpm exec hardhat deploy` runs three logical steps:

1. Lazy-deploy the deterministic-deployment-proxy CREATE2 deployer (`0x4e59b44847b379578588920ca78fbf26c0b4956c`) if not already on-chain.
2. Deploy `GPv2AllowListAuthentication` as an upgradeable proxy (separate `_Implementation` and `_Proxy` deployments).
3. Deploy `GPv2Settlement`. The `Settlement` constructor itself deploys `GPv2VaultRelayer` — the relayer address therefore must be read from the Settlement contract via `vaultRelayer()` post-deploy.

The deployer EOA controls the `manager` of `GPv2AllowListAuthentication` initially; ownership transfer is a Phase 3.x follow-up.
