## The strategic finding

CoW's settlement contract calls `authenticator.isSolver(msg.sender)` and reverts if the caller is not on a permissioned allowlist controlled by a 14-signer Gnosis Safe (`0xA03be4…`). Becoming an allowlisted solver is a multi-week governance process: forum proposal, code review, DAO vote, bonded capital. There is no technical bypass.

This means **on any chain where CoW is deployed, we cannot self-host settlement without CoW DAO approval.** The original Phase 1 spec ("self-hosted backend on Gnosis") is structurally infeasible for a no-budget bootstrap.

## Strategic pivot (decided this session)

Two parallel revenue tracks:

1. **Short-term (Phase 1.5 → 2.5):** Partner-fee injection on CoW chains. We use cowswap's existing partner-fee plumbing to capture a fixed bps. Orders flow through CoW's official orderbook, settle on CoW's contracts. We earn 75% of partner fees (CoW DAO keeps 25% per [docs](https://docs.cow.fi/governance/fees/partner-fee)), paid weekly in WETH. Covers all 10 CoW chains (Ethereum, BNB, Base, Arbitrum, Polygon, Avalanche, Linea, Plasma, Ink, Gnosis).
2. **Mid-term (Phase 3):** Own deployment on MegaETH (chain ID 4326, mainnet live since Feb 9, 2026 per [megaeth.com](https://www.megaeth.com/)). Deploy CoW's audited `GPv2Settlement` and `GPv2VaultRelayer` bytecode unchanged under our own `AllowListAuthentication` we control. We become the chain-native intent broker — first-mover on a real-time EVM with strong ecosystem reserve. The Rust services stack from Phase 1 is the production runtime.

Phase 1's vendored services stack, configs, RPC fallback, and SDK package are all preserved — they're the foundation Phase 3 builds on. **Nothing wasted.**
