## Plan deviations (documented in the validation log)

- **Chiado → Sepolia.** CoW Protocol does not run on Gnosis Chiado (404 on both prod & barn API). Sepolia was strictly better than the Gnosis-mainnet $5-cap fallback the plan suggested.
- **Programmatic swap, not browser swap.** The deployed frontend was independently verified (build green, `<title>Greg</title>` served) but the actual swap was executed via `cast wallet sign` + curl against the same CoW API the frontend calls. Browser-driven E2E lands in Phase 2 with Playwright.
- **`apps/frontend/package.json` modified to add `pnpm.onlyBuiltDependencies`.** pnpm v10's new default blocks postinstall scripts; cowswap upstream hasn't been updated. Will conflict on `git subtree pull` — track in Phase 1 sync runbook.
- **`tiny-secp256k1` patched out at Vercel build time.** Its NAN C++ addon doesn't compile on Vercel's GCC toolchain. Local builds unaffected.
- **CI: `frontend` and `backend` jobs deliberately deferred** (Phase 1) — adding them while subtrees were still in flux would have produced perpetually-red CI.
- **Vercel preview SSO disabled** so curl could probe; URL is hash-randomized but technically open. Re-enable before public Phase 2.
