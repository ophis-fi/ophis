A live, MEV-protected, intent-based swap on 11 chains — dropped into any site with a few lines. This page embeds the real widget below.

The whole integration

import { createCowSwapWidget }
  from '@cowprotocol/widget-lib'

createCowSwapWidget(el, {
  params: {
    baseUrl: 'https://swap.ophis.fi',
    appCode: 'MyDapp-via-Ophis',
    partnerFee: {
      bps: 10, // 0.10%
      recipient:
        '0x858f0F5eE954846D47155F5203c04aF1819eCeF8',
    },
    tradeType: 'swap',
    sell: { asset: 'ETH' },
    buy:  { asset: 'USDC' },
    theme: 'dark',
  },
})

React? npm i @ophis/widget-react → <OphisWidget /> injects baseUrl, appCode and pins the fee recipient for you. Full guide: docs.ophis.fi/widget.
