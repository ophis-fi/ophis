# apps/frontend build notes

Upstream: cowprotocol/cowswap (see `.greg-upstream` for pinned SHA).
Package manager: pnpm (declared as `"packageManager": "pnpm@10.30.3"` in package.json).
Workspace integration: isolated (run install/build inside apps/frontend directly; NOT merged into root pnpm workspace to avoid dep version conflicts).

## Local build
Install: `( cd apps/frontend && pnpm install --frozen-lockfile )`
Build:   `( cd apps/frontend && pnpm run build:cowswap )`
Output:  `apps/frontend/build/cowswap/` (53 MB, 268 precached entries)

## Required env vars
None required for a production build. All vars in `.env` and `.env.production` are optional (commented out).
Notable optional vars:
- `REACT_APP_INFURA_KEY` — RPC provider key (falls back to public endpoints)
- `REACT_APP_GOOGLE_ANALYTICS_ID` — analytics
- `REACT_APP_LAUNCH_DARKLY_KEY` — feature flags
- `REACT_APP_SENTRY_DSN` — error tracking

## Known gotchas
1. **pnpm v10 build-script allowlist**: pnpm v10 requires explicit opt-in for dependency postinstall scripts via `pnpm.onlyBuiltDependencies` in package.json. This field was not present upstream and was added to `apps/frontend/package.json` for this repo. The packages that need building are: `@parcel/watcher`, `@sentry/cli`, `@swc/core`, `blake-hash`, `bufferutil`, `core-js`, `core-js-pure`, `es5-ext`, `esbuild`, `keccak`, `nx`, `protobufjs`, `secp256k1`, `sharp`, `tiny-secp256k1`, `unrs-resolver`, `utf-8-validate`, `web3`, `web3-bzz`, `web3-shh`. Without this, native binaries (`nx`, `esbuild`, `@swc/core`) are not compiled and the build fails.
2. **Node version**: Upstream `.nvmrc` pins `lts/jod` (Node v22). The build was run and succeeded on Node v25.6.1 (current system). If issues arise, install Node v22 via `brew install node@22` and prefix commands with the appropriate PATH.
3. **Large chunk warning**: Vite warns that `index-ipDW0Lq9.js` is 4 MB after minification. This is a known upstream issue, not a build failure.
4. **Nx caching**: Nx caches build results in `.nx/cache/`. On subsequent runs, some tasks may be served from cache ("Nx read the output from the cache"). Force a clean build with `pnpm run build:cowswap --skip-nx-cache`.
5. **cypress excluded**: `cypress@15.4.0` is in the ignored-build-scripts list but was intentionally left out of `onlyBuiltDependencies` — Cypress is a dev/e2e tool only, not needed for the frontend build.
